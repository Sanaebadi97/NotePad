# NotePad
Vertical Lines in EditText Like NotePad


1.add jitpack to your repositories

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
	
	
  
  2.add dependencies to your build.gradle 
  
  	dependencies {
	        implementation 'com.github.Sanaebadi97:NotePad:Tag'
	}
